   Description:
This is a console-based Rock Paper Scissors game developed using C. It features two exciting modes:
 **3-Player Mode:** Play with two of your friends locally.
 **Human vs Computer Mode:** Test your luck against the computer with an automated scoring system.

The game includes colorful text outputs and sound effects (Beep sounds) to enhance the user experience.

## How to run the project:
1. Ensure you have a C compiler installed on your computer (e.g., CodeBlocks, Dev-C++, or GCC).
2. Download the `main.c` file from this repository.
3. Open the file in your compiler/IDE.
4. Build and Run the project.
   *(Note: This project uses <windows.h>, so it is designed to run on Windows OS.)*
